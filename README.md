# LINE BOT 智能聊天機器人

### 說明
使用OpenAi - text-davinci-003 串接 LineBot 製作出一個小型聊天機器人，
使使用者可在Line上即可跟chatgpt互動，而不用至網頁上使用。

| Python | Flask |
